# MNE-BIDS-Pipeline [2]: Sensor analysis

This Brainlife App runs the sensor-level analysis of MNE-BIDS-pipeline on the derivatives produced by the previous preprocessing App (MNE-BIDS-Pipeline [1]: Preprocessing). It computes condition contrasts, decoding and time frequency analyses for EEG and MEG data, covering the following steps:
1. Takes the preprocessed BIDS derivatives generated by the preprocessing App as input.
2. Computes condition contrasts when defined.
3. Runs decoding/MVPA when contarsts are provided, including:
   1. Time-resolved decoding.
   2. Temporal generalization.
   3. Common spatial patterns (CSP).
   4. Bootstrap resampling.
4. Runs time-frequency analysis (power estimation) over a configurable frequency range.
5. Generates HTML reports summarizing the sensor-level results.

# Input 

The BIDS derivatives folder produced by the M/EEG Preprocessing App (bids_dir) with a config.json generated by Brainlife that sets the pipeline parameters.

# Output

- out_dir: including a copy of the derivatives plus the sensor-level analysis outputs generated by MNE-BIDS-Pipeline.
- html_report

# How to run this App on Brainlife 

The App runs inside a Singularity container built from ghcr.io/carlota-juarez/app-mne-bids:lite, based on python:3.11-slim with mne, mne-bids, and mne-bids-pipeline installed. These are the steps to run the App:
1. Register on Brainlife.io and create a project.
2. Select this App from the App page. Set the output of the previous App as the input and configure the provided parameters to suit the selected data.
Execute the App. Once completed, the results can be dwonloaded and passed to the Source Analysis App.
